vim
===

## Installation
```
# Switch directory to home
cd ~
# Clone git repo
git clone git://github.com/davidsteinsland/vim.git
# Rename repo to .vim
mv vim/ .vim
# Move vimrc file to ~/.vimrc
mv .vim/vimrc .vimrc
# Create tmp and backup dirs
mkdir .vim/tmp && mkdir .vim/backup
````

### Windows specific
* Replace `~/.vim/` with `%HOME%\vimfiles`
